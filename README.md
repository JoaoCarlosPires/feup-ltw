## Features
 - Security
     - XSS: yes
     - CSRF: yes
     - SQL using prepare/execute: yes
     - Passwords: bcrypt 
     - Data Validation: php
 - Technologies
     - Separated logic/database/presentation: yes
     - Semantic HTML tags: yes
     - Responsive CSS: yes
     - Javascript: yes
     - Ajax: yes
     - REST API: no 
  - Usability:
     - Error/success messages: yes, in some cases
     - Forms don't lose data on error: no
